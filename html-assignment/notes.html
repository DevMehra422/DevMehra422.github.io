Tags That I Have Learnt

<img> tag:
<img> tag is used link a image to a web page from a another source.This tag can also be given "alt" attribute which describe the context of the image if the image is not loaded.
<br> tag:
A <br> tag is used to add a single line break after a piece of code.It is a self-closing tag which means it does not require a closing tag
<strong> tag:
<strong> tag is widely used for specifying "strong importance" in text.Majority of the times it is used inside <p> tag or paragraph tag.
<hr> tag:
<hr> tag is used for adding a horizontal line across the webpage, it is commonly used to create division between the content of the website.This tag is widely used by online newspaper websites.
<body> tag:
<body> tag is one of the most important tag in a HTML web page as it contains all the tags which are used to build a HTML webpage.Anything written outside the <body> tag would not be considered by HTML or the webpage itself will not load.The <body> tag is the foundation block for a HTML webpage.
